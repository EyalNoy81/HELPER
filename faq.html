שאלות נפוצות - דואר אלקטרוני

איך מגדירים חשבון דואר חדש?
1. פתח את אפליקציית הדואר
2. לחץ על "הוסף חשבון"
3. הכנס את כתובת המייל והסיסמה
4. עקוב אחר ההוראות להשלמת ההגדרה
איך משחזרים סיסמה לחשבון דואר?
1. לחץ על "שכחתי סיסמה" בדף ההתחברות
2. הזן את כתובת המייל שלך
3. בדוק את תיבת הדואר החלופית
4. עקוב אחר ההוראות בהודעה שקיבלת
כיצד מגדירים חתימה אוטומטית?
1. פתח הגדרות
2. בחר ב"חתימה"
3. הכנס את הטקסט הרצוי
4. שמור את השינויים

שאלות נפוצות - וואטסאפ

איך מגבים הודעות בוואטסאפ?
פתח הגדרות > צ'אטים > גיבוי צ'אטים > גבה עכשיו. הגיבוי יישמר בענן או במכשיר.
איך משתמשים בוואטסאפ ווב?
1. היכנס ל-web.whatsapp.com
2. פתח את וואטסאפ בטלפון
3. סרוק את הקוד QR
4. המכשיר יחובר אוטומטית
כיצד מפעילים אימות דו-שלבי?
1. פתח הגדרות
2. בחר "חשבון" > "אבטחה"
3. הפעל אימות דו-שלבי
4. הגדר קוד PIN

שאלות נפוצות - פייסבוק

איך משנים סיסמה בפייסבוק?
היכנס להגדרות > אבטחה והתחברות > שנה סיסמה. תצטרך לאמת את זהותך לפני השינוי.
כיצד מגבילים את חשיפת הפרופיל?
1. פתח הגדרות פרטיות
2. בחר "מי יכול לראות את התוכן שלי"
3. התאם את ההגדרות לפי העדפותיך
4. שמור שינויים
איך מדווחים על תוכן פוגעני?
1. לחץ על שלוש הנקודות ליד הפוסט
2. בחר "דווח על פוסט"
3. בחר את סיבת הדיווח
4. שלח את הדיווח
